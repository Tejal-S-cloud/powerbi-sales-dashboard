# 📊 Power BI Sales Dashboard

## 📌 Overview
This project presents an interactive sales dashboard built using Power BI. It helps analyze sales performance, cost, and profit across different regions and product categories.

## 🚀 Features
- KPI Cards (Sales, Cost, Profit)
- Sales by Country (Map)
- Sub-category analysis
- Regional distribution (Pie Chart)
- Time-based trend analysis
